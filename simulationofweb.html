WebSimulation using html,css &js

Web simulations

Web simulation is the invocation of computer simulation services over the world wide web ,specifically through the web browser.

Unfortunately, these are bulky Java programs that require downloading to view them. Therefore two web-based versions were written in JavaScript to give easy access to these programs.

Scientific paper

This project started as an assignment project, so naturally there is a paper on the subject which explains the background, function, and applications of computer simulations.

The entire project is about web simulations.
